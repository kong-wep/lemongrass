# Lemongrass

Website to host my blog

## Installation

To install the project, run

    npm install

To run the project, run

    npm run start

To build the project, run

    npm run build

## Project Structure

Structure is a typical Astro project.

* Code is in /src
* Builds to /dist
* Images is in /public
* Blogs & Book notes in /src/content
